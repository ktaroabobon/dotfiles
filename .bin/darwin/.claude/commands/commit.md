すべての変更を commit してください。
commit 以外のことはしないでください。

## commit 前の必須チェック
以下のコマンドを実行して、CIチェックを事前に通過することを確認してください：
1. `make ci` - backend のローカルで CI をチェックするコマンド
2. その他プロジェクト固有のチェックコマンド

問題が見つかった場合は修正してから commit してください。

## commit メッセージのルール
commit メッセージを作成するときは以下に絶対従ってください：
- 日本語で書くこと
- 端的に書くこと
- Claude に関する記述を含めないこと
- 一括ではなく塊ごとに commit すること
