# dotfiles

## Overview

This repository contains my dotfiles.

## Requirements

- MacOS version 11.0 or later
- Ubuntu version 22.04

## Installation

Download this repository:

```sh
cd ~ && git clone https://github.com/ktaroabobon/dotfiles.git
```

### MacOS
Set up for MacOS:

```sh
cd dotfiles 
make all
```

### Ubuntu
Set up for Ubuntu:

```sh
cd dofiles
.bin/init.sh
```
